# TechShop - Nền tảng TMĐT Công nghệ tích hợp AI

## Stack
- Backend: Spring Boot 3 + PostgreSQL 16 + Redis 7
- Frontend: Next.js 14 + TypeScript + Tailwind CSS
- AI Gateway: Node.js 20 + Groq API
- Deploy: VPS Ubuntu 2GB RAM

## Structure
- /backend      - Spring Boot Modular Monolith (:8080)
- /frontend     - Next.js App Router (:3000)
- /ai-gateway   - Node.js AI proxy service (:3001)
- /docs         - Tài liệu, Postman collections
- /docker       - Docker Compose configs
